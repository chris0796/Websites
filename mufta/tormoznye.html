 ОБЩАЯ ИНФОРМАЦИЯ
 Электромагнитные муфты ЭТМ предназначены для организации дистанционного управления приводами металлообрабатывающих станков, а так-же другого промышленного оборудования различного назначения. Основная функция электромагнитных муфт тормозного исполнения - это автоматизация процесса остановки и последующего удержания рабочего вала приводного механизма.

 Маркировка электромагнитных муфт заключает в себе следующую информацию:
 Муфта ЭТМ-ХХ16-Х2Х3, где:
       ЭТМ - серия электромагнитных муфт (аналогичные муфты Э1ТМ, Э11М и ЕТМ);
        ХХ1 - габарит муфты (см. "Инфо" для каждой муфты);
          6 - исполнение муфты - "Масляная", "Тормозная";
         Х2 - размер посадочного отверстия (см. "Инфо" для каждой муфты);
         Х3 - группа посадочного отверстия (см. "Инфо" для каждой муфты).

 Масляные электромагнитные муфты тормозного исполнения серии ЭТМ состоят из следующих основных элементов:
 - Втулка. Внутренняя поверхность втулки, для закрепления на валу приводного механизма, выполнена в соответствии с группой и размером посадочной характеристики: шпонка или шлицы. Наружная поверхность, для зацепления со внутренними фрикционными дисками муфты, выполнена в виде эвольвентных шлицев. 
 Втулка в тормозных муфтах серии ЭТМ "существует" отдельно от самой муфты. При монтаже, сначала на рабочем валу приводного механизма закрепляют втулку, а потом уже на нее устанавливают муфту. Замена вышедшей из строя муфты производится не затрагивая установленной втулки. Данное свойство конструкции тормозных муфт позволяет, при проведении ремонтных работ, заменять муфты, в пределах одного габарита, без учета их посадочных характеристик. К примеру, вышедшую из строя муфту ЭТМ056-2Н, при условии сохранения на валу ее установочной втулки, можно заменить муфтой ЭТМ056-1А.
 - Корпус. Для установки в приводном механизме, корпус муфты изготавливается двух видов: с квадратным фланцем (крепежные отверстия в углах фланца) и без фланца (крепежные отверстиями на поверхности корпуса). Внутри корпуса установлена электромагнитная катушка, оба конца которой выведены через специальное отверстие наружу. Для фиксации наружных фрикционных дисков, в корпусе муфты предусмотрен стопорный поводок.
 - Фрикционный пакет. Чередующиеся между собой внутренние и наружные магнитопроводящие фрикционные диски. Внутренние диски закрепляются на эвольвентных шлицах установочной втулки, наружные диски, с помощью выступов, закрепляются в стопороном поводке корпуса муфты.
 - Якорь. Якорь представляет собой диск, изготовленный из магнитной стали.

 При выключенной электромагнитной катушке, внутренние фрикционные диски, закрепленные на установочной втулке, вращаются вместе с рабочим валом приводного механизма, в котором используется муфта. Наружные диски, закрепленные в стопорном поводке корпуса муфты, находятся в неподвижном состоянии.
 При подаче напряжения на катушку, возникает электромагнитная сила, притягивающая якорь к корпусу муфты. Притянувшийся якорь сжимает фрикционный пакет, прижимая наружные фрикционные диски ко внутренним, вследствие чего между ними возникает сцепление и внутренние диски, прекращая вращение, тормозят рабочий вал приводного механизма с моментом, равный вращающему моменту муфты. По окончанию торможения, вал удерживается с моментом, равным передаваемому моменту муфты.
 Расцепление фрикционного пакета, при выключении муфты, обеспечивается специальной волнообразной формой внутренних дисков, при этом якорь отходит до упора в ограничительное кольцо стопорного поводка.

 Подача электрического тока на катушку, для включения муфты, осуществляется по проводам выведенным через специальное отверстие в корпусе муфты.
